# UNLIMITR AI Cognitive Backbone v1.0

The brain layer for the Genie AI Health Dashboard. Importable behavioral + scoring + narrative framework.

## Modules

| Module | Purpose |
|--------|---------|
| `healthScoring.js` | 5-dimension model: Metabolic, Hormonal, Behavioral, Recovery, Risk Momentum |
| `behaviorClassifier.js` | Type A–E psychology: High Motivation/Low Discipline, Emotional Eating, Overtrained, Start-Stop, High Compliance |
| `narrativeEngine.js` | 7-part report template: Emotional Alignment → Biological → Root Cause → Reversibility → Timeline → Confidence |
| `riskGradient.js` | Ambient bands: Deep Metabolic Red, Adaptive Amber, Stabilizing Teal, Structured Green, Resilient Emerald |
| `planRecommendation.js` | Score-based plan logic without direct selling |

## AI Identity

- Clinical but compassionate
- Data-backed but emotionally intelligent
- Never: aggressive, fear-based, salesy, shame, panic
- Always: empowering, clarity, reversibility, structure, guidance

`assets/data/AI_Knowledge_Base.json` — Condition intelligence (thyroid, PCOS, diabetes), tone rules, narrative templates.
