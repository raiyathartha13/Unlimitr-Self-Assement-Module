# Body Models for Transformation Animations

## 2D Images (Fat→Fit morph)

| File        | Use                           |
|-------------|-------------------------------|
| `3.png`     | Male body — Fat→Fit slider    |

Add `female.png` for female users; both use `3.png` until then.

## 3D GLB Models (body-morph.js)

| File                      | Use          |
|---------------------------|--------------|
| `character__male_01.glb`  | Male 3D body |
| `stylized_lowpoly_female.glb` | Female 3D body |
| `male_avatar.glb`         | Alternative male |
| `male_lowpoly_basemesh.glb` | Alternative male |

The morph-engine uses images for the "See Your Transformation" slider. The body-morph module (when used) loads GLB models by gender from this folder.
